# Online Shopping System

## Overview

This project is a Java Swing-based Online Shopping System created for Secure Coding Review. The application demonstrates secure login, product recommendation, billing workflow, and secure payment processing.

## Features

* User Login Authentication
* Recommended Products
* Add to Cart
* Billing System
* Secure Payment Approval
* Card Number Masking
* Interactive GUI using Java Swing

## Technologies Used

* Java
* Java Swing
* AWT Event Handling

## How to Run

Compile the program:

```bash
javac OnlineShoppingSystem2.java
```

Run the program:

```bash
java OnlineShoppingSystem2
```
