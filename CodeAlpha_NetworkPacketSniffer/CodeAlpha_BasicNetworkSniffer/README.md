# CodeAlpha Network Packet Sniffer

## Overview

The CodeAlpha Network Packet Sniffer is a Python-based cybersecurity project developed using the Scapy library. This project captures and analyzes live network packets to understand how data travels through a network.

The program displays useful packet information such as:

* Source IP Address
* Destination IP Address
* Protocol Type
* Packet Details

This project helps beginners learn packet analysis, network monitoring, and basic cybersecurity concepts.

---

## Features

✔ Capture live network traffic packets
✔ Display source and destination IP addresses
✔ Detect TCP, UDP, and ICMP protocols
✔ Analyze packet structure and payload data
✔ Understand network communication flow

## Installation

### Step 1: Install Python

Download Python from:
https://www.python.org/downloads/

---

### Step 2: Install Scapy

Open Command Prompt and run:

```bash
pip install scapy
```

---

### Step 3: Install Npcap

Download and install Npcap from:
https://npcap.com/

While installing, enable:

* Install Npcap in WinPcap API-compatible Mode
