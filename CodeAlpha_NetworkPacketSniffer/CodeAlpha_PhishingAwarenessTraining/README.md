# 📖 Project Description

This project is an interactive cybersecurity awareness website designed to educate users about phishing attacks, fake websites, social engineering techniques, and online safety practices.

The module helps users understand how cybercriminals trick people into sharing sensitive information such as passwords, OTPs, banking details, and personal data.

It also includes an interactive quiz system and a cybersecurity chatbot for better user engagement and learning.
